# Budget_Tool

A tool to manage my budget records kept on Google Sheets.

## Installation

Clone the project into your working directory:

```bash
$ git clone https://github.com/kmhoran/budget_tool.git .
```

## Usage

Build and run

```bash
$ dotnet build
$ cd AppCli
$ dotnet run
```
